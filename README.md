## 设计模式概述

设计模式是程序员在面对同类软件工程设计问题时所总结出来的有用的经验。模式不是代码，而是某类问题的通用解决方案，设计模式代表了最佳的实践。设计模式的本质是提高软件的可维护性、通用性和可扩展性，并降低软件的复杂度。

设计模式可以分为三大类，共 23 种：
